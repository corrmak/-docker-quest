Решение 4 алгоритмических задач и веб-приложения на C#

## Задача 1: Разложение на простые множители

**Условие:**  
Разложить натуральное число на простые множители.

**Формат входных данных:**  
Одно натуральное число N, где 2 ≤ N ≤ 2147483647

**Формат выходных данных:**  
Простые множители в порядке неубывания через пробел.

**Примеры:**

Например:

На вход поступает число 30
На выходе должно получиться: 2 3 5 (то есть 2*3*5 = 30) Причём 2,3,5 все простые числа. то есть каждое делится только на нуль и на себя. (3,10 не подойдет, т.к. 10 не простое число)
______________________
На вход поступает число 123368748
На выходе должно получиться 2 2 3 19 71 7621 
______________________
На вход поступает число 45645645664
На выходе должно получиться 2 2 2 2 2 1426426427 
______________________

## Задача 2: Следующее число с различными цифрами

**Условие:**  
Найти минимальное число, которое строго больше заданного и имеет все различные цифры.

**Примеры:**
________________________
Входные данные
1987
Выходные данные
2013 (т.к. 2013 это минимальное число после 1987, где все цифры разные)
________________________
Входные данные
2014
Выходные данные
2015
_______________________
Входные данные
5555555
Выходные данные
5601234
_______________________

## Задача 3: Цифровой корень

**Определение:**  
Цифровой корень — это однозначное число, полученное путем итеративного сложения цифр исходного числа.

**Пример вычисления:**

Например цифровой корень 65536 это 7, потому что 6+5+5+3+6=25 и 2+5=7.
__________________
Входные данные
144213
Выходные данные
6
_________________
Входные данные
654545
Выходные данные
2

**Условие:**

**Примеры:**
_____________________
Входные данные
